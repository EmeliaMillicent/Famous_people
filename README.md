# Famous_people
This project, detailed in the famous_people.sql file, is a comprehensive SQL solution for creating, populating, and querying a database about famous individuals in the entertainment industry. It features a well-designed relational schema across five tables (Famous_People, Movies, Songs, Awards, Regions) and includes ten analytical SQL queries. These queries extract insights on topics like individual influence, success metrics, geographical impact, industry trends, and demographic analyses. The project is a practical resource for learning SQL, data modeling, and basic data analysis, with the added benefit of being suitable for visualization in tools like Power BI.
