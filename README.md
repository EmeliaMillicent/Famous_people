#  Famous People SQL Project

This project (`famous_people.sql`) is a complete **SQL solution** for creating, populating, and analyzing a database of famous individuals in the entertainment industry. It’s designed for learning SQL, data modeling, and basic data analysis, with outputs that can easily be used in **Power BI** or **Excel** dashboards.  

##  Project Overview
- **Database Design:** 5 relational tables — `Famous_People`, `Movies`, `Songs`, `Awards`, and `Regions` — capturing personal details, career achievements, and geographic info.  
- **Data Population:** Includes realistic sample data to enable robust practice and testing.  
- **Analytical Queries:** 10 advanced SQL queries that provide insights on:
  -  Individual influence and popularity  
  -  Career success metrics (awards, hit movies/songs)  
  -  Geographical impact and regional trends  
  -  Industry-wide trends and comparisons  
  -  Demographic analyses (age, gender, region)  

##  Learning Goals
- Design and normalize a relational database.  
- Practice **SQL DDL & DML commands** (CREATE, INSERT, UPDATE, DELETE).  
- Perform **analytical queries** with joins, aggregations, filters, and ranking.  
- Prepare query outputs for **data visualization** in Power BI or Excel.  

##  Practical Applications
- Build interactive **dashboards** or **reports** using query outputs.  
- Explore **trends in the entertainment industry** or perform data-driven storytelling.  
- Understand how to **model complex relationships** and extract meaningful insights from data.  

This project is a hands-on resource for anyone looking to **strengthen SQL skills** and **gain insights from structured entertainment datasets**.
